Qué son las aplicaciones progresivas, conocidas habitualmente como Progressive Web Apps o por sus siglas PWA. Un nuevo tipo de aplicaciones web que nos permiten realizar muchas de las cosas que antes sólo se podía desde aplicaciones instaladas mediante las store de apps.

¿Quieres conocer cuáles serán las webs del futuro? En el video de hoy te voy a hablar de las aplicaciones web progresivas, que nacen de la mezcla de una App y una Web.

¿Pero que pasaría si estas aplicaciones web se pudieran ejecutar como si fueran aplicaciones locales? Está tecnología se conoce como Aplicaciones Web Progresivas, o PWA por sus iniciales en inglés, mira el video para que descubras de que se trata.
